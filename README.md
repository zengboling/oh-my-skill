# oh-my-skill 🚀

A curated library of AI skills, procedural guides, and specialized workflows for AI agents.

## 📁 Structure
- `skills/`: The core library of skills, organized by category.
- `templates/`: Templates to ensure consistency across all skills.

## 🛠 How to add a new skill
1. Use the `templates/SKILL_TEMPLATE.md` as a starting point.
2. Create a new folder or file under `skills/` based on the category.
3. Document the trigger conditions, exact steps, and common pitfalls.
4. Commit and push to GitHub.
